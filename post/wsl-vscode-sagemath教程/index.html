wsl+vscode+sagemath教程

2 分钟·

1.安装wsl
#

网上教程挺多的，随便搜一下就有，就不赘述了

我选用的是微软商店搜到的第一个Ubuntu，其他的也都可以

2.安装anaconda
#

网上搜索anaconda下载linux版（miniconda也可以）Download Now | Anaconda

在我的电脑输入\\wsl$或者直接进入ubuntu的文件夹（如果有）

把刚才下载好的文件，如Anaconda3-2024.10-1-Linux-x86_64.sh复制过去，进行安装

bash Anaconda3-2024.10-1-Linux-x86_64.sh

后边基本上yes和回车即可

可以用conda info检测一下是否安装到位

3.创建sagemath虚拟环境
#

这里前边我下载的anaconda带的是3.12的python环境，安装的时候看自己的版本情况

conda config --add channels conda-forge
conda create -n sage sage python=3.12

一段时间后会有这个界面

这就装好了

4.vscode远程连接wsl
#

安装插件wsl

然后连接ubuntu

创建一个文件夹用与存放sage文件然后打开

安装SageMath Enhanced插件

然后就可以运行了，写个test测试一下

注意：

启动点击小三角运行会有command not found，这是因为sage环境没有激活，使用：

conda activate sage

(先点击小三角，到新生成的终端内再激活)

到此就可以了

5.优化
#

1.安装库可以用

conda install -n sage [package]

2.每次运行后都会有一个py文件，可以再写一个脚本（sage文件）删一下，如：清除py文件.sage

import os
import glob

def delete_specific_files(directory, file_extension):
    pattern = os.path.join(directory, f'*{file_extension}')

    files_to_delete = glob.glob(pattern)

    for file_path in files_to_delete:
        try:
            os.remove(file_path)
            print(f"文件 {file_path} 已成功删除。")
        except Exception as e:
            print(f"删除文件 {file_path} 时发生错误: {e}")

directory_to_search = ''
file_extension_to_delete = '.py'
delete_specific_files(directory_to_search, file_extension_to_delete)

3.可以在~/.bashrc文件末尾加一行

conda activate sage

达到默认启动虚拟环境的效果，不用再激活环境
